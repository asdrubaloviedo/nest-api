## GitHub

```bash
$ git clone git@github.com:asdrubaloviedo/nest-api.git
```


## Installation

```bash
$ npm install
```

## Running the DB
```bash
$ docker compose up dev-db -d
$ docker ps
$ docker logs CONTAINER-ID-NUMBER-FROM-DOCKER-PS
$ npx prisma migrate dev
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
